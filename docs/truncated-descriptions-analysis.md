# Truncated Descriptions Analysis

## Background

The v2 description run (`outputs/v2/image_analysis.json`) used `max_tokens=80`. Out of 31,841 entries, **9,301 hit the token limit** — meaning generation stopped before the model naturally concluded.

These truncated entries are collected in `outputs/v2/truncated_descriptions.yaml`, generated by `scripts/find_truncated_descriptions.py`.

## Original Hypothesis (Revised)

The initial assumption was that truncation indicated a poor photo — the model rambled because there was nothing to focus on. This turned out to be **wrong**.

Comparing sharpness distributions across all 31,841 images:

| | Median sharpness | % below 100 |
|---|---|---|
| Truncated (n=9,301) | **938.9** | 10.1% |
| Non-truncated (n=22,535) | 629.5 | 15.6% |

Truncated images are actually **sharper** on average. The model hit the token limit because photos had more content to describe — busy scenes, crowds, sports action — not because they were featureless or blurry. Blurry photos may produce *shorter* descriptions because the model has less to say.

## Sharpness Scoring

Laplacian variance scores were computed from the 512x512 thumbnails (note: NEF thumbnails are JPEG-encoded with the original filename preserved). Scores are stored in:
- `outputs/v2/sharpness_scores.json` — truncated set only (9,301 images)
- `outputs/v2/sharpness_scores_all.json` — full collection (31,841 images)
- `image_sharpness` table in `mediameta.db` — truncated set, importable via `scripts/import_sharpness_scores.py`

A sharpness viewer for the 500 lowest-scoring truncated images is at `outputs/v2/sharpness_viewer.html` (generated by `scripts/generate_sharpness_viewer.py`). Visual inspection confirmed the low-scoring images include: genuine blur, extreme close-ups, motion blur, plain/featureless shots, and lens flare.

## ISO Is the Strongest Sharpness Predictor

Among truncated photos with camera data, ISO correlates most strongly with Laplacian sharpness:

| ISO | n | Median sharpness |
|---|---|---|
| ≤ 100 | 4,162 | 1,419.9 |
| 101–400 | 1,671 | 921.7 |
| 401–1600 | 2,938 | **349.5** |
| > 1600 | 96 | 529.6 |

High ISO = low light = camera shake or subject motion = blur. Photos at ISO 401–1600 score less than a quarter of low-ISO shots.

Wide aperture (f/1–f/2) also correlates with lower sharpness due to shallow depth of field:

| Aperture | n | Median sharpness |
|---|---|---|
| f/1–f/2 (very wide) | 499 | 371.1 |
| f/2–f/4 | 4,307 | 1,266.4 |
| f/4–f/8 | 3,185 | 710.4 |
| f/8+ (narrow) | 876 | 446.4 |

The f/8+ drop is likely due to diffraction at very narrow apertures rather than focus issues.

Shutter speed showed no strong pattern:

| Shutter speed | n | Median sharpness |
|---|---|---|
| < 1/500s (fast) | 4,290 | 926.9 |
| 1/500–1/100s | 2,358 | 1,124.0 |
| 1/100–1/30s | 971 | 763.8 |
| > 1/30s (slow) | 1,248 | 800.1 |

Fast and mid-range shutters score similarly. Slow shutters (> 1/30s) score lower as expected, but the effect is weaker than ISO.
